# Smart Solution - 병동 민원 관리 대시보드

병원 병동별 솔루션 관련 민원을 접수하고, 실시간으로 현황을 모니터링하는 관리자/간호사용 웹 대시보드입니다.

## 주요 기능

- **간호사 페이지**: 병동별 민원 접수, 솔루션/상태별 통계 시각화
- **관리자 페이지**: 전체 민원 현황, 병동/솔루션별 분석, 실시간 폴링, 계정 관리
- **인증 시스템**: Access Key 기반 로그인, 역할별 접근 제어 (admin/nurse)

## 기술 스택

| 기술 | 용도 |
|------|------|
| Flask | 백엔드 웹 프레임워크 |
| SQLite | 데이터베이스 |
| Jinja2 | HTML 템플릿 |
| Chart.js | 차트 시각화 |
| Werkzeug | 비밀번호 해싱 |

## 실행 방법

```bash
pip install flask
python app.py
# http://localhost:5001 접속
```
